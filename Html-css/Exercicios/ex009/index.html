Trabalhando com Listas

Listas Ordenadas

Nas listas ordenadas existem 5 tipos de ordem, que são: 1, A, a, I, i

Para colocar os tipo de ordem nas listas basta colocar o atributo type dentro da Tag OL

  1. acordar
  2. comer
  3. treinar
  4. ler
  5. trabalhar
  6. estudar

Listas não Ordenadas

Nas listas não ordenadas (UL) também existem3 tipos de ordem, que são: Disc, Circle e Square.

  • Pão
  • ovos
  • manteiga
  • açucar
  • arroz
  • café
