Principais formatação

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag b (não semântica) Ou seja, não tem significao, apenas forma.

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica)

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

DICA: Evite usar a tag N & I por ser da época do HTML4.

Então é recomendado usar as tag STRONG & EM.

Atalho pra "envelopar" rapidamente.

CTRL, ↑ +P ; (Seleciona a lista de Comandos)

ENTER, N, I, P, H2, ... (TAG DESEJADA) + ENTER (Pra Selecionar a tag)

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK

E no outro parágrafo, temos texto marcado no final.

OBS: Para selecionar e mudar todas as marcações, basta colocar STYLE dentro do documento

Texto grande e Texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

A tag BIG pra colocar o texto grande está obsoleta

Texto deletado

Podemos marcar um texto como excluído para indicar que ele pode ser lido mas não considerado.

Usando a tag DEL

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que foi adicionado depois.

Usando a tag INS

Existe também o sublinhado com a tag U (não semântica)

basta colocar a tag SUP

Texto subscrito

Para inserir coisas do tipo H2O
