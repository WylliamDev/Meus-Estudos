Usando links externos

Você pode acessar o meu repositório público no GitHub

Você também pode acessar o meu canal no YouTube

Para usar links externos e abrir uma nova aba, basta usar a Tag target="_blank" rel="external" dentro da Tag a href=

Target="_blank" - Abre uma janela em branco

Rel="external" - Vai pra fora do site

  • Target
    • Target="_blank"- Abre uma janela em branco
    • Target="_self" - Abre a guia do link na mesma página
  • Rel
    • Rel="external"- Vai pra fora do site
    • rel="_prev" - Volta pra página anterior
    • rel="_next" - Vai pra próxima página
